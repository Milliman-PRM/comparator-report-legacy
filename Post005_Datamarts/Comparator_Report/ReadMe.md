## Comparator Report

The contents of this data mart will be delivered to support the Comparator Report for New York Premier.

### Contents

| Table | Contents |
| :--- | :------- |
| `cost_util` | Aggregate cost and utilization information |
| `cost_util_benchmark` | Loose and well managed benchmarks |
| `memmos` | Aggregate member months |
| `memcnt` | Counts of unique members by dimensions |
| `ref_prm_line` | Reference data set describing PRM lines. |
| `details_inpatient` | Decorated IP Cost Model Aggregation. |
| `details_snf` | Decorated SNF Cost Model Aggregation |
| `metrics_key_value` | Key-value store of metrics |
| `meta_project` | Key-value store of project metadata. |
| `meta_field` | Key-value store of field level information. |
| `time_windows` | Reference table explaining the claims periods included in each time period. |
